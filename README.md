# 🛒 GoCart - Full Stack E-Commerce Platform

A modern full-stack e-commerce web application built using **Next.js**, **React.js**, **Prisma**, and **Tailwind CSS**. The platform provides a seamless shopping experience with responsive UI, dynamic routing, product browsing, search functionality, and shopping cart management.

---

## 🚀 Features

- 🛍️ Browse products with responsive product cards
- 🔍 Search products instantly
- 🛒 Shopping Cart functionality
- 📄 Dynamic Product Details Page
- 📱 Fully Responsive Design
- ⚡ Fast Navigation with Next.js App Router
- 🗂️ Modular Component-Based Architecture
- 🗄️ Database Integration using Prisma ORM

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/zustnithin/gocart3.git
```

Move into the project

```bash
cd gocart3
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
